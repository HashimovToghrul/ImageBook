# ImageBookApp

A simple iOS app to save and display user photos using Core Data.

## Overview
This app lets users save images with titles and display them in a list.  
It practices UIKit, Core Data, UIImagePickerController, and basic MVC/MVVM.

## Tech
- Swift
- UIKit
- Core Data

## Features
- Add photo with name, artist, year
- View saved photos in a list
- Detail view for each saved item
- Delete items

## Usage
Clone the repo, open in Xcode, and run on a simulator or real device.
